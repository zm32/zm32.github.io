Was ist Django Girls Bern

Django Girls Bern ist ein Workshop wo man Programmieren lernt.

Waehrend zwei Tagen machen wir eine eigene Webseite, mit einem Blog. Dazu verwenden wir verschiedene Tools, Frameworks und Programmiersprachen. Ein paar Beispiele sind:

  • HTML
  • CSS
  • GIT
  • Python
  • Django
  • Command line

Alle sind super motiviert und die Coaches haben viel geduld. Zudem gibt es feines Essen.
